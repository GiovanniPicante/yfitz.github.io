Was ist JavaScript?

  • Entwickelt von Brendan Eich als "LiveScript" in 1995 und umbenannt in "JavaScript"
  • Standardiziert von ECMAScript Spezifikationen. ES5 (in 2009)
  • Eine client-seitige Sprache. Der Browser liest den Code und führt in direkt aus.
  • Interfaces mit HTML & CSS lassen uns dynamische Webseiten entwickeln, die auf Benutzereingaben reagieren.

Wie funktioniert JavaScript?

  1. Du besuchst eine Website mit Javascript im Code.
  2. Dein Browser (e.g., Chrome) liest den Code Zeile für Zeile.
  3. Der Browser führt jede Zeile direkt aus, die er liest.
  4. Basierend auf den Anweisungen, führt der Browser Kalkulationen und Veränderungen am HTML und CSS durch.
  5. Wenn der Browser eine Anweisung nicht versteht, erzeugt er eine Fehlermeldung.

Variablen

Variablen sind Container für Werte, die sich ändern.

Deklarieren und Initialisieren von Variablen

Um eine Variable zu deklarieren (erzeugen) einfach var und den Variablenname.

var numberOfKittens;

Der Variable einen Anfangswert geben - heisst Initialisierung.

var numberOfKittens = 5;

Variablen Werte

  • Variable (nach der Deklaration) ohne Wert ist undefined.
  • Variablen können auf einen Wert gesetzt werden.
  • Variablen können unterschiedliche Typen von Daten enthalten.
  • Der Wert einer Variable kann sich ändern.

Namen von Variablen

  • Der Name von Variablen ist case-sensitive (unterscheidet Gross- und Kleinbuchstaben).
  • Der Variablenname muss eindeutig sein
  • Der Variablenname muss mit einem Buchstaben beginnen, $, oder _.
  • Vermeide reservierte Wörter..
  • Aussagekräftige Variablennamen sind wichtig für das Verständnis ihres Codes.

Verwendung von Variablen

Nach der Variablendeklaration, kann diese im Code verwendet werden.

var numberOfKittens = 5;
console.log(numberOfKittens);

Datentypen

  • string Zeichenketten 
    var userName = 'Jane Lane';
  • number Ganze- oder Gleitkommazahl 
    var myAge = 30;
  • boolean true oder false 
    var catsAreBest = true;
  • undefined kein Wert zugewiesen 
    var favoriteThings;
  • null leerer Wert 
    var goodPickupLines = null;

Zahlen

Variablen können Zahlenwerte haben, entweder Ganzezahlen (integer) oder Dezimalzahlen (floats).

var numberOfKittens = 5;
var cutenessRating = 9.6;

JavaScript wandelt integer automatisch in float um.

NaN = Not-A-Number

Arithmetische Operatoren

Beispiel Name Ergebnis
-a Negation Gegenteil von a.
a + b Addition Summe a und b.
a - b Subtraction Differenz von a und b.
a * b Multiplikation Product von a und b.
a / b Division Quotient von a und b.
a % b Modulus Rest bei Division von a und b.
++a Increment um 1 Summe von a und 1.
a-=b Differenz Differenz von a und b a zugewiesen.

Strings

Variablen können Zeichenketten(strings) enthalten. Die Zeichenketten stehen in einfachen oder doppelten Anführungszeichen.

var kittensName = 'Fluffy';

If you want to use a quote in your string, you'll need to escape it with a backslash.

console.log('I\'d like to use an apostrophe');

Zeichenketten Operatoren

Zeichenketten zusammenfügen mit a +, der concatenation Operator.

var kittensName = 'Fluffy ';
var fullName = kittensName + 'McDougle';
console.log(fullName); // Outputs 'Fluffy McDougle'

An Zeichenkette am Ende anfügen mit +=.

var kittensName = 'Admiral ';
kittensName += 'Snuggles';
console.log(kittensName); // Outputs 'Admiral Snuggles'

Kombiniere Zeichenketten und Zahlen

In Konkatenation von Zeichenketten und Zahlen, werden Zahlen wie Zeichenketten behandelt.

var numberOfFruit = 6;
var typeOfFruit = 'bananas';
var allTheFruit = 'I have ' + numberOfFruit + ' ' + typeOfFruit + '!';
console.log(allTheFruit);
