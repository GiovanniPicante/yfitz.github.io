Kapitel 3

Arrays

Array ist eine Liste von Elementen.

var arrayName = [value0, value1];

Die Element können unterschiedlichen Datentyp haben.

var rainbow = ['Red', 'Orange', 'Yellow', 'Green', 'Blue'];

var lotteryNumbers = [33, 72, 64, 18, 17, 85];

var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];

Länge und Elemente

Die Methode length zählt die Elemente.

var rain = ['Red', 'Orange', 'Yellow', 'Green','Blue'];
console.log(rainbowColors.length);

Zugriff auf Elemente mit [] und Index.(beginnt bei 0)

var firstColor = rainbowColors[0];
var lastColor  = rainbowColors[4];

Verändern von Array Elementen

rain[3] = "White";

Verlängern von Arrays

Arrays haben keine feste Länge. Einfach durch push können diese verlängert werden.

var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];

myFavoriteThings.push('Dancing');

Schleifen durch Arrays

for-Schleife durchläuft Array.

var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green','Blue'];
  
for (var i = 0; i < rainbowColors.length; i++) {
  console.log(rainbowColors[i]);
}

Alternativ foreach-Schleife

array.forEach(function(arrayElement) { 
                console.log(arrayElement); 
});

Beispiel oben als foreach:

rainbowColors.forEach(function(arrayElement) {
  console.log(arrayElement);
});

Array Funktionen

  • push (Element anhängen) a b c<- d
  • pop (Element abhängen) a b c-> d
  • unshift (Element vorne anhängen) a->b c d
  • shift (Element vorne abhängen) a<- b c d
  • reverse (Umkehren) d c b a
  • sort (Array sortieren) a b c d
  • concat (zwei Arrays zusammenfügen)
  • slice (Teile ermitteln)
  • splice (Elemente entfernen , einfügen)

Objekte

Objekte lassen uns eine Menge von Eigenschaften speichern.

var objectName = {
  propertyName: propertyValue,
  propertyName: propertyValue
};


var user = {
  hometown: 'Atlanta, GA',
  hair: 'Auburn',
  likes: ['knitting', 'code'],
  birthday: {month: 10, day: 17}
};

Zugriff auf Objekte

Mit der Punkt Notation kann man auf das Objekt zugreifen.

var user = {
  hometown: 'Atlanta, GA',
  hair: 'Auburn'
};

var usersHometown = user.hometown;

oder mit der Klammer Notation (so wie bei Arrays)

var usersHair = user['hair'];

Objecte verändern

Mit beiden Notationen kann man die Eigenschaften eines Objekts verändern.

var user = {
  hometown: 'Atlanta, GA',
  hair: 'Auburn'
};

user.hair = 'blue';

Neue Eigenschaften hinzufügen

user.married = true;

oder löschen.

delete user.married;

Arrays von Objekten

Weil Arrays verschiedene Datentypen enthalten können, können diese auch Objekte enthalten.

var users = [
  {name: 'Jolene', age: 21},
  {name: 'Alexa',  age: 18}
];

for (var i = 0; i < users.length; i++) {
  var user = users[i];
  console.log(user.name + ' is ' + user.age + ' years old.');
}

Objekte

Objekte können auch als Argumente von Funktionen übergeben werden.

var jolene = {
  age: 21,
  hairColor: 'Auburn',
  likes: ['pizza', 'tacos'],
  birthday: {month: 3, day: 14, year: 1995}
}

function describeUser(user) {
  console.log('You are ' + user.age + ' years old with '
  + user.hairColor + ' hair.');
}

describeUser(jolene);

Objekt Methoden

Objekte können auch Methoden enthalten.

var jolene = {
  age: 21,
  hairColor: 'Auburn',
  talk: function() {
    console.log('Hello!');
  },
  eat: function(food) {
    console.log('Yum, I love ' + food);
  }
};

Methodenaufruf mit der Punkt-Notation:

jolene.talk();

jolene.eat('pizza');
