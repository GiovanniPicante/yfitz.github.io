Deklaration von Funktionen

Um eine Funktion zu deklarieren schreibt man Code in{}

function parrotFacts() {
  console.log('Some parrot species can live for over 80 years');
  console.log('Kakapos are a critically endangered flightless parrot');
}

Argumente

Funktionen akzeptieren Eingabewerte, sogenannte Argumente.

function callKitten(kittenName){
  console.log('Come here, ' + kittenName + '!');
}

callKitten('Fluffy'); // outputs 'Come here, Fluffy!'

function addNumbers(a, b){
  console.log(a + b);
}

addNumbers(5, 7);  // outputs 12
addNumbers(9, 12); // outputs 21

Rückgabewerte

Werte, die beim Aufruf an die aufrufende Funktion zurückgegeben werden.

function square(num) {
  return num * num;
}

console.log(square(4));       // outputs '16'

var squareOfFive = square(5); // squareOfFive equals '25'

return beendet die Funktion augenblicklich.

Global Scope

Eine Variable, die ausserhalb von einer Funktion deklariert wurde, hat einen global scope und auf diese kann von überall zugegriffen werden.

var awesomeGroup = 'Girl Develop It'; // Global scope

function whatIsAwesome() {
  console.log(awesomeGroup + ' is pretty awesome.'); // Will work
}

whatIsAwesome();

Local Scope

Eine Variable, die in einer Funktion deklariert wurde, hat einen local scope und auf diese kann nur innerhalb der Funktion zugegriffen werden..

function whatIsAwesome() {
  var awesomeGroup = 'Girl Develop It';              // Local scope
  console.log(awesomeGroup + ' is pretty awesome.'); // Will work
}

whatIsAwesome();

console.log(awesomeGroup + ' is pretty awesome.'); // Won't work

Boolean Variablen

Boolean Variables sind die Werte true and false

var catsAreBest = true;
var dogsRule = false;

Neben true oder false, können auch andere Werte im boolean Kontext ausgewertet werden. Folgende werden zu false ausgewertet: falsy 

// Diese Variable wird zu false ausgewertet

var myName = '';
var numOfKids = 0;
var isMarried;     // Variable ohne Werte ist undefined

null und NaN wird zu false ausgewertet.

Alles andere wird zu true ausgewertet.

if statement

Verwende if statements um durch eine Bedingung Ausführung von Code zu steuern.

if (condition) {
  // statements to execute
}

var age = 30;

if (age > 18) {
  console.log('You are an adult');
}

Comparison Operators

Example Name Result
a == b Equal TRUE if a is equal to b (can be different types).
a === b Identical TRUE if a is equal to b, and the same type.
a != b Not equal TRUE if a is not equal to b (can be different types).
a !== b Not identical TRUE if a is not equal to b, or they are not the same type.
a < b Less than TRUE if a is strictly less than b.
a > b Greater than TRUE if a is strictly greater than b.
a <= b Less than or equal to TRUE if a is less than or equal to b.
a >= b Greater than or equal to TRUE if a is greater than or equal to b.

if/else statement

Alternative Code wird mit else erzeugt.

var age = 30;

if (age >= 16) {
  console.log('Yay, you can drive!');
} else {
  console.log('Sorry, you have ' + (16 - age) +
  ' years until you can drive.');
}

if/else statement

Mehrere Alternativen mit else if.

var age = 30;

if (age >= 35) {
  console.log('You can vote AND run for President!');
} else if (age >= 30) {
  console.log('You can vote AND run for the Senate!');
} else if (age >= 18) {
  console.log('You can vote!');
} else {
  console.log('You can\'t vote, but you can write your representatives.');
}

Logical Operators

Example Name Result
a && b And TRUE if both a and b are TRUE.
a || b Or TRUE if either a or b is TRUE.
!a Not TRUE if a is not TRUE.

Using logical operators

You can use these operators to combine conditions.

var age = 30;
var yearsAsCitizen = 30;

if (age >=30 && yearsAsCitizen > 9) {
  console.log('You can run for the Senate!');
} else {
  console.log('You are not eligible to run for the Senate');
}
